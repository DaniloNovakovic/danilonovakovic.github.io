## Outcome

Round 001 selected **Sketchbook Ridge** as the global winner. The production
plan became **Sketchbook Ridge Summit**:

- Ridge provides the playable overworld spine.
- Summit provides the production and architecture discipline.
- Archipelago contributes the visible sticker / world-memory reward layer.
