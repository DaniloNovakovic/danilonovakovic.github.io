# Project Assets & References

This folder contains the design references and prompts for the "Digital Sketchbook" portfolio project.

## Image Prompt for Assets

If you need to generate new character or building assets, use this prompt:

> *"Create 2D character sprites in the Open Peeps style. Hand-drawn ink aesthetic, monochromatic black and white, diverse modern clothing, expressive but simple facial features. Perspective: Profile/Side-view. White background."*

## Reference Aesthetics

- **Pablo Stanley's Open Peeps:** Modular, clean character design.
- **Life is Strange (Diary/Journal):** Hand-drawn ink aesthetic on high-quality paper.
- **Strict Monochromatic:** High contrast B&W with sketchy cross-hatching.
