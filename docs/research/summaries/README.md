# Research Summaries

This folder contains browseable summaries and comparison notes that can still
inform Sketchbook Ridge design.

- `game-inspirations/`: existing-game fun analyses and transferable design
  lessons.
- `design-theory/`: level design, playability, combat, and map-design theory.
- `production/`: production process, architecture quality, and scope-management
  research.

These notes are not specs. Treat them as idea material unless a current
`docs/game-design/` document adopts the idea.
