# Research

This folder stores historical and reference research for Sketchbook Ridge.
Current direction still lives in `docs/game-design/`, `.agents/skills/`, and
`.agents/rules/`.

## Folder Map

- `summaries/`: browseable research summaries and reusable inspiration notes.
- `provenance/`: source research, deep reports, and historical rationale that
  informed active docs but should not be treated as current spec.

Load active game-design docs, agent skills, and scoped rules first. Use research
when Danilo asks for rationale, comparison material, deeper theory, or a fresh
synthesis pass.
