# Research Provenance

This folder stores source research and background reports that informed active
Sketchbook Ridge design, but are not the current spec.

Agents should not load provenance reports by default. Start from the active
design docs, skills, and scoped rules; use provenance only when Danilo asks for
source rationale, comparison material, or a fresh synthesis pass.

## Subfolders

- `agents/`: deep research reports that informed specialist agent/skill design.
- `audio/`: background audio research and Audio Designer provenance.
- `narrative/`: background narrative and tribute writing research.
- `visual/`: background visual-direction, asset-pipeline, and style-system
  research.
